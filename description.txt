Problem: Regional Sales Summary Generator

Objective:
Write a Python function named `generate_sales_summary` that reads a CSV file containing sales data, calculates total sales per region, identifies the top 3 regions by total sales, and writes the results to an output CSV.

Input CSV:
- File name: problems/regional_sales_summary/input_data.csv
- Columns:
    - OrderID (string): Unique sale identifier
    - Region (string): Geographic region
    - SalesPerson (string)
    - Product (string)
    - UnitsSold (integer)
    - UnitPrice (float)

Requirements:
1. Read the CSV file from the path: 'problems/regional_sales_summary/input_data.csv'
2. Calculate total sales for each row: UnitsSold * UnitPrice
3. Aggregate total sales by region
4. Rank regions from highest to lowest total sales
5. Select the top 3 regions
6. Write output to 'outputs/output_summary.csv' with columns: Region, TotalSales, Rank
7. Skip rows with missing or non-numeric UnitsSold or UnitPrice
8. Only the provided input CSV should be used

Example input (CSV table format):

OrderID | Region | SalesPerson | Product | UnitsSold | UnitPrice
1001    | North  | John        | Widget  | 5         | 20.0
1002    | South  | Alice       | Gadget  | 3         | 50.0
1003    | East   | Bob         | Thingamajig | 5      | 15.0
1004    | North  | John        | Widget  | 2         | 20.0
1005    | South  | Alice       | Gadget  | 0         | 50.0
1006    | East   | Bob         | Thingamajig | 0      | 15.0
1007    | South  | Charlie     | Widget  | 0         | 50.0

Expected output (outputs/output_summary.csv):

Region | TotalSales | Rank
South  | 150.0      | 1
North  | 140.0      | 2
East   | 75.0       | 3

Important:

- The function must write the summary CSV to 'outputs/output_summary.csv'.
- Additionally, the function must return the list of dictionaries containing the top 3 regions.
- Each dictionary should have the keys: 'Region', 'TotalSales', 'Rank'.
- Example return value:

[
    {"Region": "South", "TotalSales": 150.0, "Rank": 1},
    {"Region": "North", "TotalSales": 140.0, "Rank": 2},
    {"Region": "East", "TotalSales": 75.0, "Rank": 3}
]
